<h2>Introduction</h2>
온라인 쇼핑이 활성화 되고 있는 상황에서 안경 및 선글라스 제품의 구매에 불편함을 겪는 소비자들에게 상품의 이미지, 사이즈 등을 '안경상점'앱을 통해 미리 써보며 구매에 도움을 주고자 한다.
